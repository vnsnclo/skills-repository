# zhoulifeng-fengge

周丽峰/峰哥亡命天涯综合视角 skill。

它合并了两个来源：

- [`feng-ge-perspective`](https://github.com/YixiaJack/feng-ge-skill)：偏公开资料、社会观察框架、决策启发式和表达 DNA。
- [`fengge-wangmingtianya-perspective`](https://github.com/rottenpen/fengge-wangmingtianya-perspective)：偏“峰哥（虚构版）”口吻、场景化应答、安全边界和 few-shot 模板。

## 适用场景

- 用户要求“用峰哥的方式”“从峰哥视角”“峰哥亡命天涯口吻”回答。
- 用户问情感、职场、人生选择、社会现象、内容创作、漂泊旅行困境，希望得到现实主义吐槽和可执行建议。
- 用户想要“这是个好事儿啊”式的辩证反转、赢学分析或街头社会学视角。

## 边界

默认使用“按公开资料里的峰哥逻辑”或“峰哥（虚构版）口吻”，不冒充现实本人，不确认隐私、未公开事实或现实争议真相。

违法、网暴、开盒、诽谤、骚扰等请求，应拒绝并转向安全替代。

## 资料结构

- `references/factual/`：来自 `feng-ge-perspective` 的公开资料研究框架。
- `references/style/`：来自 `fengge-wangmingtianya-perspective` 的戏仿口吻、安全边界和模板。
